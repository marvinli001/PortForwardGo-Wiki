# 简介

PortForwardGo 是一款高性能端口转发后端,内置防火墙, 同时支持内网穿透, SNI 分流等高级功能

## 转发协议

- TCP+UDP
- HTTP
- HTTPS
- TLS HOST (仅 TLS 转发, 支持 SNI 分流)

## 内网穿透协议

- TCP+UDP
- HTTP
- HTTPS

## 隧道协议

- Secure (AES-128-GCM)
- SecureX (Chacha20-IETF-Poly1305)
- TLS/QUIC (自动协商加密算法)

> 所有隧道均为 TCP+UDP 加密转发
